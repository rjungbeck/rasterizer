Welcome to Rasterizer.

Rasterizer allow you to convert a PDF into PNG files. It alos allows the conversion of PNGs to printer specific specific PRN files using WIndows printer drivers.

Rasterizer has a command line and a pipe interface.

This project is maintained by rjungbeck
